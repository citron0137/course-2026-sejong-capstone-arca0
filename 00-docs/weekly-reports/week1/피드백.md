# 피드백 정리

## 우리 팀 피드백

1. 12주차까지 개발이 끝나도록 일정 수정 필요
2. 프로젝트 제안서에 팀 번호 잘못 작성되어있음

## 게임 개발 팀 피드백

1. 게임 개발이 산으로 가는 경우가 더러 있음<br>=> 시놉시스와 같이 기준점이 잘 정해져있어야한다 
2. 게임 개발이 완성되지 못하는 경우도 더러 있음<br>=> 매 주차 진행을 잘 보여줘야할듯

## 다른 팀 피드백
1. 페이지 번호를 꼭 넣자 (PPT에도)
2. PPT를 만들 때는 내용 누락을 최소화하자
3. 제안서에도 도표와 시각 자료를 넣어 가독성을 높이자 
4. 시스템 구성도와 개발 환경 구체적으로 작성 필요
